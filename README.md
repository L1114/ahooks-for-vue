Vue Hooks 库

## 📚 文档

- [查看文档](https://l1114.github.io/ahooks-for-vue/)

## ✨ 特性

- 易学易用
- 包含大量提炼自业务的高级 Hooks (开发中)
- 包含丰富的基础 Hooks (开发中)
- 使用 TypeScript 构建，提供完整的类型定义文件 (开发中)
- 支持 vue2 / vue3

## 📦 安装

```bash
$ npm install --save pxhooks
# or
$ yarn add pxhooks
# or
$ pnpm add pxhooks
# or
$ bun add pxhooks
```

## 🔨 使用

```js
import { useRequest } from "pxhooks";
```
