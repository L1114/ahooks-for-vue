## 📚 文档

- 开发中

## ✨ 特性

- 易学易用
- 包含大量提炼自业务的高级 Hooks
- 包含丰富的基础 Hooks

## 📦 安装

```bash
$ npm install --save pxhooks
# or
$ yarn add pxhooks
# or
$ pnpm add pxhooks
# or
$ bun add pxhooks
```

## 🔨 使用

```js
import { useRequest } from "pxhooks";
```

打开浏览器访问 http://127.0.0.1:8000
